# 2026-05-09 工作日志

## 更新日志超链接支持
- 修改 `ChangelogItem.jsx`，使用 `dangerouslySetInnerHTML` 渲染更新日志文本，支持 HTML 标签（如 `<a>` 超链接）
- 在 `index.css` 添加 `.changelog-link` 样式，链接在浅色/深色模式下均有适配样式（蓝色下划线）

## 重大更新通知功能
- 新建 `ChangelogNotification.jsx` 组件，实现右下角弹窗提示未读重大更新
  - 使用 localStorage (`changelog_lastViewedMajorVersion`) 记录用户已查看的重大更新版本
  - 仅检测 `type === 'major'` 的版本更新
  - 页面加载后延迟 2 秒显示通知（滑入动画）
  - 点击通知跳转到更新日志并标记已读；关闭按钮可暂时隐藏
  - 版本号语义化比较（支持 `x.y.z` 格式）
- 在 `App.jsx` 中集成组件：页脚"查看更新日志"链接也同步调用 `markChangelogAsRead()`
