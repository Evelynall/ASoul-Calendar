# 自定义颜色功能说明

## 功能概述
现在您可以为每个成员（角色）自定义背景颜色，让日程表更符合您的个人喜好。

## 使用方法

### 1. 打开设置页面
点击右上角的设置图标（齿轮图标）进入设置页面。

### 2. 找到"外观选项"区域
在设置页面中找到"外观选项"部分，其中包含：
- 显示模式设置
- 组合配色开关
- **自定义成员颜色**（新增功能）

### 3. 自定义颜色
在"自定义成员颜色"区域中：
- 每个成员都有一个颜色选择器（色块）
- 点击色块可以打开颜色选择器
- 选择您喜欢的颜色后，该成员的所有日程背景色会立即更新
- 颜色代码会显示在成员名称下方

### 4. 恢复默认颜色
如果想要恢复所有成员的默认颜色：
- 点击"恢复默认"按钮
- 确认后所有自定义颜色将被清除，恢复为原始配色

## 支持的成员
- 贝拉
- 嘉然
- 乃琳
- 思诺
- 心宜
- A-SOUL
- 小心思
- 其他

## 技术细节
- 自定义颜色保存在浏览器的 localStorage 中
- 即使刷新页面，您的自定义颜色也会保留
- 自定义颜色会覆盖默认配色，但不会影响原始配置
- 支持所有标准的 HTML 颜色格式（十六进制颜色代码）

## 注意事项
- 自定义颜色仅在当前浏览器中生效
- 如果清除浏览器数据，自定义颜色会被重置
- 建议选择对比度较高的颜色以确保文字清晰可读
